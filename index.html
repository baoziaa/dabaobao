山叶口景区

山叶口景区的自然风光非常不错，这里是一处以山水、地貌风光为主的自然风景区，有着独特的五彩石地貌，还有古松等等，在景区内还有一座规模很大的关帝庙，游客们可以进去参拜祈福。

青山关

青山关是长城上修建的一座古城堡，它始建于明朝万历年间，是由抗倭名将戚继光主持修建。这座城是名副其实的古城，城内还建有许多明朝的小院子，显得十分气派。

唐山抗震纪念馆

唐山抗震纪念馆展示着唐山的过往，它是由9个部分组成，充分的回顾了创业历程，告慰罹难同胞，追忆八方恩情，教育后世子孙，激励全市人民为建设经济发展、社会和谐的新唐山而努力奋斗的主题思想。

Button Button

运河唐人街不夜城

运河唐人街不夜城最妙的就是它的夜景，这里的建筑是仿明清的古建筑，它还结合了现代的功能，将唐山传统的戏曲文化、茶文化和饮食文化融入其中，是人们娱乐、购物的大型原始复古商业街。

Button Button

景忠山

景衷山是唐山当地的一座名山，山间的古迹众多，随处即可看见道教、佛教、儒家等教别的建筑。此外，山间的树木葱郁、景致秀美，尤其是山间的古松形态各异，很适合拍照摄影。

Button Button

开滦国家矿山公园

开滦国家矿山公园是一座以煤矿文化为主题的公园，这里向广大的游客展示着煤矿历史、采煤过程和历史上的煤矿罢工运动等事迹。在这里可以了解到清末洋务运动的知识，煤矿的历史知识等等，内容十分的丰富。

Button Button

南湖公园

南湖公园是一座综合性的公园，其可供市民游憩和进行水上活动。这片公园的绿化做的相当的不错，湖水、绿地、森林、花草共同组合了天然的生态景观，如同江南水乡一般显得婀娜多姿。

Button Button

清东陵

清东陵是清朝帝王后妃的陵墓群，它是我国现存规模最大、体系最为完整的皇陵建筑群，还被列入了世界文化遗产的名录当中。传闻当年的顺治帝在这一带打猎时被其钟灵敏秀所震撼，于是就在这块风水宝地上建成了这片气势恢宏的清东陵。

Button Button

滦州古城

滦州古城是在原有的滦州衙门上重新修建的，古城内林立着许多古色古香的店铺，人们可以在街上观看表演、品尝美食、拍照留影。格外值得一说的是滦州古城里的美食，这里不仅有当地的特色食物，还汇集了全国各地的著名小吃。
